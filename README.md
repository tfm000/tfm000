### What I'm building 🔭
 
🔹 **[CopulAX](https://github.com/tfm000/copulax)** — JAX-based probability modelling library with an emphasis on low-dimensional optimization. Univariate, multivariate, and copula distribution fitting with JIT compilation and autodiff support. [![PyPI](https://img.shields.io/pypi/v/copulax)](https://pypi.org/project/copulax/) [![build](https://github.com/tfm000/copulax/actions/workflows/tests.yml/badge.svg?branch=main)](https://github.com/tfm000/copulax/actions/workflows/tests.yml)
 
🔹 **[Diana](https://github.com/tfm000/diana)** — Locally hosted text-to-speech document converter. Turns PDFs, EPUBs, webpages, and news feeds into MP3 audio using local (Kokoro, Piper) or cloud (OpenAI, ElevenLabs) TTS engines. Includes an AI-powered news aggregator and LLM text cleaning. Built with Streamlit.
 
🔹 **[SklarPy](https://github.com/tfm000/sklarpy)** — Copula fitting in Python. Implements Gaussian, Student-T, Skewed-T, Generalized Hyperbolic copulas and 100+ univariate distributions via scipy.stats, with integrated parameter fitting and visualisation. [![PyPI](https://img.shields.io/pypi/v/sklarpy)](https://pypi.org/project/sklarpy/)
